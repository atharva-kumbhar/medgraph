System Architecture

Three distinct pipelines.

1

LLM Only

Direct query to the hosted LLM. No retrieval or external context.

2

Basic RAG

SentenceTransformer embedding → FAISS vector retrieval → LLM generation.

3

GraphRAG

Entity extraction → TigerGraph direct-neighbor retrieval → ranked graph chunks → LLM generation.
